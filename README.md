# myStatBot

Этот бот помогает регулярно вести личную статистику и отображает корреляцию значений.
- Добавить метрику
- Добавить значение метрике
- Настроить метрику
- Экспорт в .csv
- Построить график корреляции метрик
- Удалить все сохраненные метрики
